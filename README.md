# IOTA Wallet for iOS

The IOTA Wallet is a WiP App, use it at your own risk.

## Install

`pod install`

## Features

- Get balance from seed (text)
- Get balance from seed (QRCode)
- Get transaction history
- Generate new address (attach to tangle)
- Local PoW
- Send transaction
